# Phonebook API

Built using
1. .NetCore 3.1
2. MySQL
3. MediatR
4. AutoMapper
5. JWT Auth
6. Swagger
7. Uses the concept of Domain Driven Design

## Execute

```bash
docker-compose up
```

## Access API Swagger

Browse to http://localhost:5000/swagger

## Access Database

Browse to http://localhost:6001

    User: user
    Password: password
